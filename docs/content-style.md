# Content Style

## Core Writing Principles

- Keep the original meaning.
- Keep the scripture logic.
- Keep the theological center.
- Use natural Chinese, not translationese.
- Prefer clear and warm language over argumentative language.
- Avoid unnecessary absolutes such as “一定”, “绝对”, “毫无争议”, unless the source and scripture clearly require them.
- Avoid adding claims that are not present in the source.

## Translation Fidelity

When source material needs translation, translation fidelity overrides style polish.

This applies especially to:

- Sermon PDFs
- Sermon manuscripts
- Bilingual Japanese/English manuscripts
- English articles
- Japanese materials
- Any other foreign-language source

Rules:

- Translate sentence by sentence according to the original meaning.
- Do not replace translation with summary, outline, thematic rewrite, or devotional expansion.
- Preserve all arguments, examples, scripture references, quotations, transitions, repeated emphasis, tone, and conclusions.
- Natural Chinese word order is allowed, but information must not be removed.
- Do not merge multiple original paragraphs in a way that loses detail or rhetorical flow.
- Do not add applications, explanations, or theological claims that the source did not make.
- If the source is bilingual, confirm which language version is the translation source. If the user did not specify, state which version is being followed before translating.

For sermons, the final Chinese TXT must be a faithful complete translation and correction of the source, not a short sermon summary.

## Faith Article Style

For articles intended as Chinese faith essays:

- Use a calm, reflective tone.
- Make transitions explicit.
- Prefer readable paragraph length.
- Explain background when it helps ordinary readers.
- Keep scripture citations close to the point they support.
- Keep challenge and exhortation, but avoid harsh or accusatory language unless the source is intentionally direct.

## Public Platform Style

For 公众号 or broadly public contexts:

- The tone should be warm, measured, and readable.
- Avoid unnecessary culture-war framing.
- Avoid over-direct institutional language when softer wording can carry the same meaning.
- Use “我们” carefully; do not sound superior to the reader.
- End with reflection, invitation, or a grounded application rather than a slogan.

## Paragraph Formatting

When the user asks for a publishable article draft outside the website import scripts:

- Use clear headings.
- Keep paragraphs short.
- If requested for WeChat/公众号, each paragraph may begin with two full-width spaces.
- Do not force two full-width spaces into Markdown frontmatter or website-generated content unless explicitly requested.

## Safer Wording Examples

Use only when suitable for the target platform:

- “教会” -> “群体 / 信仰群体”
- “讲道” -> “分享 / 信息”
- “罪” -> “偏离 / 破碎 / 深处的问题”
- “悔改” -> “转向 / 重新回到正确方向”

These are style choices, not theology replacements. If a biblical passage directly uses “罪” or “悔改”, keep the theological meaning clear.

## Quality Checklist

Before delivering or publishing:

- Title matches the scripture and article focus.
- Summary is complete and does not stop mid-sentence.
- Scripture references are accurate.
- Category is correct.
- Translation did not omit examples or transitions.
- Generated Markdown has required frontmatter, including `articleId` when used by the site.
- The live page, if published, contains expected title/category/body text.
